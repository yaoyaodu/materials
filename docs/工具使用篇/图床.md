# 图床

**图床**一般是指**储存图片的服务器**，有国内和国外之分。国外的图床由于有空间距离等因素决定访问速度很慢影响图片显示速度。国内也分为单线空间、多线空间和cdn加速三种。

PicGo简单点说就是一个**图床上传工具**，我们主要就是借助这个软件进行上传。
